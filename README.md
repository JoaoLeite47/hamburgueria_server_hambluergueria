# API Hambluergueria

API desenvolvida utilizando Nest.js.

## Como fazer o clone

```bash
git clone https://github.com/JoaoLeite47/hamburgueria_server_hambluergueria.git
```

## Como instalar as dependências

```bash
npm install
```

## Como rodar o projeto

### Produção

```bash
npm run start
```

### Desenvolvimento

```bash
npm run start:dev
```
